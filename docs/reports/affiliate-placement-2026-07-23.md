# 问题文档：联盟位置与零转化

记录时间：2026-07-23
相关：`docs/tasks/04-affiliate-links-revenue-integrity.md`（Task 04，状态 partially complete）、`docs/reports/gsc-ga-follow-up-2026-07-23.md`

## 一句话

联盟链接放在页面 85–90% 深处，而访客平均只停留 16 秒；过去 28 天 `affiliate_click` 触发 **0 次**。但把链接往前挪会直接撞上本站赖以立身的克制原则，而且按当前流量算，就算挪对了也几乎挣不到钱。**所以这既不是一个纯技术问题，也不是一个现在就该动手的问题。**

## 一、实测事实

### 1.1 转化为零

GA4 property `p536218791`，过去 28 天收到的事件只有 6 个：

```
click, first_visit, page_view, scroll, session_start, user_engagement
```

`affiliate_click` 不在其中。在事件列表里搜 `affiliate` 返回 0 项。

副作用：GA4 不允许给"从未接收过"的事件标记关键事件（页面横幅要求点击事件名旁边的星号，而它根本没有那一行）。所以**转化追踪也无法预先配置**，只能等第一次真实点击出现后再补。

### 1.2 追踪代码没有问题

排查过，不是 bug：

| 检查项 | 结果 |
|---|---|
| 构建产物含 `data-analytics-event="affiliate_click"` | 108 个页面 |
| `layouts/partials/affiliate-footer.html` 输出 partner / slot 属性 | 是 |
| partial 被布局引用 | 4 处：`is-ivisa` / `country-form` / `how-to-fill` / `single` |
| `assets/js/analytics.ts` 监听 `a[data-analytics-event="affiliate_click"]` | 是 |
| SafetyWing URL 已配置 | 是 |

补充：gtag 代理曾有 bug（推数组而非 `arguments`），已由 `52d084d`（2026-07-15）修复，`check-seo-output.mjs` 也加了断言守住。所以 7-15 之后的零点击是真实的。

### 1.3 位置

联盟区在成品页里的相对位置（按 HTML 字节偏移量计算，是渲染滚动深度的**近似**，不是精确像素深度）：

| 页面 | 联盟区字节位置 | 页面大小 |
|---|---:|---:|
| `/indonesia/e-cd/` | 全文 85% 处 | 38 KB |
| `/malaysia/mdac/` | 全文 86% 处 | 40 KB |
| `/singapore/how-to-fill/` | 全文 90% 处 | 57 KB |

渲染条件是 `{{- if .Section }}`，所以首页不渲染，实际覆盖 108 个页面。因为 `airalo_url = ""`，两栏网格里**当前只有 SafetyWing 一张卡**。

### 1.4 访客根本走不到那里

同期 GA4（28 天）：

| 指标 | 值 |
|---|---:|
| 活跃用户 | 462 |
| 新用户 | 460（几乎无回访） |
| 平均互动时长 | **16 秒** |
| 跳出率 | 66.9% – 94.1% |

流量来源：`chatgpt.com / ai-assistant` 203 人（44%）、直接访问 230 人、`google / organic` 仅 21 人。

这批人的行为是明确的：在 ChatGPT 里问"印尼入境卡官网是哪个"，拿到链接过来核对一眼官方域名，然后走。**核对一个 URL 本来就是秒级动作。** 他们不是来读长文的，自然也到不了 85% 深处。

## 二、机制

不是"卡片不够吸引人"，而是**卡片和访客意图不在同一个时间尺度上**：

- 访客的任务：确认一个政府网址是不是真的 → 10 秒内完成 → 离开
- 卡片的位置：文章末尾，需要读完整篇字段指南才会遇到
- 卡片的文案：`Support this guide` / `If you still need coverage for the trip...` —— 刻意写得很轻，是"顺手支持一下"，不是产品推荐

三者叠加，0 次点击是可预期的结果，不是异常。

## 三、真正的张力

这才是需要你拍板的部分，不是技术问题。

### 3.1 站点已经公开承诺了这个模式

- 每篇文章侧栏（`rail_money_explainer`）：**"Clearly disclosed travel affiliate links at the bottom of this article."**
- `content/about.md`：**"That's it. That is the entire business model."**，并强调"We have **never** taken money from iVisa or any visa middleman. We will not, ever."

Task 04 当初把联盟位做出来，一半原因就是**堵住可信度缺口**——文中反复说"我们靠联盟链接吃饭，所以没有动机推你去中介站"，如果读者滚到底发现根本没有链接，整套论证就垮了。文档原话：最坏情况是读者认定"整个站是表演性质的，从此不再相信我们关于哪个政府域名是真的判断"。

**换句话说：联盟位的第一职能是让诚信叙事成立，第二职能才是赚钱。** 这一点在评估任何改动时不能忘。

### 3.2 Task 04 明确写了不许做什么

`docs/tasks/04` 的 "Out of scope (do not do)" 是硬约束，不是建议：

- 不加展示广告（AdSense 等）
- 不超过两个联盟伙伴（"more dilutes per-partner conversion and looks spammy"）
- 不在正文里硬塞联盟链接；**"The footer is the right place."** 内联提及只在**上下文真的成立**时可以
- **不做弹窗、退出挽留浮层、"等一下再走"遮罩**；原话："The site's minimalism is a competitive advantage. Don't burn it."

任何"把卡片挪到首屏""加个浮动条""滚动到一半弹出来"的方案，都直接违反已经写下的决定。要做就得先明确推翻它，而不是绕过去。

### 3.3 按当前流量，改位置也换不来钱

这是最容易被忽略、但最该先算的一笔账。用当前数据做量级估算（假设已标注，非精确预测）：

- 28 天 462 活跃用户
- 假设把卡片可见率从现在的极低值提到 **20%**（乐观）→ 约 92 次曝光
- 这种"顺手支持"型卡片的点击率现实区间 **0.5%–2%** → **0.5–2 次点击/月**
- 保险类联盟从点击到成交的转化率通常是低个位数百分比 → **约 0.02–0.06 单/月**

即便位置优化做到 10 倍效果，月成交仍然远小于 1 单。

**结论：当前的约束是流量，不是位置。** 在 462 用户/月的量级上，为了提升联盟转化去动那套让站点可信的克制设计，是拿最稀缺的资产（可信度）去换一个数学上兑现不了的收益。

## 四、我们还不知道的（这决定了下一步该测什么）

现在无法区分两种情况：

- **A. 卡片从没被看到**（可见性问题）
- **B. 卡片被看到了但没人想点**（吸引力/意图问题）

A 和 B 的应对完全不同，而现在只有"点击=0"这一个数，不足以区分。

**最便宜的判别方法**：GA4 的增强衡量会在滚动到 **90% 深度**时触发 `scroll` 事件，而联盟区正好起始于 85–90%。所以：

```
scroll 事件数 / page_view 事件数 ≈ 有多少次浏览真的滚到了联盟区
```

`scroll` 已经在收集了（在近期事件列表里）。**去 GA4 拉这两个数的比值，就能判定是 A 还是 B**，不需要写任何代码。建议在做任何位置决策前先取这个数。

## 六、需要谁决定什么

这不是能靠数据自动得出的结论，因为它涉及一个价值判断：**为了让一个当前几乎产生不了收入的位置更有效，愿意消耗多少"这个站不像在推销"的观感。**

需要拍板的三件事：

1. 现在是否值得投入（结合 3.3 的测算，倾向是"不"，但这是你的判断）
2. 如果要动，是否推翻 Task 04 的 out-of-scope 条款；推翻的话要写进 Task 04 并注明理由
3. Airalo 要不要补齐——这件事独立于位置问题，Task 04 本来就没做完

## 七、建议的最小下一步

无论最终选哪个方向，**先取 `scroll / page_view` 比值**（第四节）。花几分钟，能把"看不到"和"不想点"分开，避免在错误的假设上做设计决策。
