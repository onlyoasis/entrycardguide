## 简短回答

iVisa 不是 Visit Japan Web。Visit Japan Web 是免费的日本政府服务。签证服务、电子签证和入境二维码流程是不同东西。

如果付费站说可以替你处理 "Japan entry"，先问清楚它到底提交什么：签证申请、Visit Japan Web 二维码流程，还是只给一份付费清单。

## 怎么核对

1. 官方来源在 `.go.jp` 域名下。
2. Visit Japan Web 免费。
3. 签证服务不能替代 Visit Japan Web 二维码设置。
4. 账号最好由你自己掌握。
